# Fight - 人机吵架决策系统

一个通过人机吵架来帮助做出重大决策的CLI应用。

## 功能特点

- 🗣️ **智能辩论** - 与AI智能体进行结构化辩论
- 💾 **共享记忆** - 使用agents_memory存储论点和证据
- 🎯 **决策辅助** - 基于辩论内容生成决策建议
- 📊 **论证追踪** - 记录所有论点和证据来源

## 安装和使用

### 环境设置

```bash
# 克隆仓库
git clone git@github.com:rebeccaz4/fight.git
cd fight

# 安装依赖
pip install -r requirements.txt
```

### 配置agents_memory

确保已设置agents_memory环境变量：

```bash
export AGENTS_MEMORY_DSN="postgresql://am_app:password@localhost:5432/agents_memory"
export AGENTS_MEMORY_API_KEY="sk_am_your_demo_key"
```

### 运行吵架应用

```bash
python -m fight.cli
```

## 使用流程

1. **开始辩论**: 输入决策主题和初始立场
2. **展开论证**: 用户与AI轮流提出论点和反驳
3. **证据记录**: 系统自动保存重要论点和证据
4. **决策生成**: 基于辩论内容生成最终决策建议

## 技术栈

- **CLI界面**: 使用Python的input/print和rich库提供美观的终端界面
- **AI智能体**: 使用Anthropic Claude API进行智能对话
- **记忆存储**: agents_memory PostgreSQL数据库
- **逻辑处理**: 本地Python逻辑进行论点分析
